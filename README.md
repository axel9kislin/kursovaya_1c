# kursovaya_1c
Курсовая работа по 1С для Василия Андреевича.

## Небольшая инструкция по тому, как развернуть данную конфигурацию, и какие сторонние программы нужны для её исполнения.
1) В правом верхнем углу зелёная кнопка clone or download, выбрать скачать как zip. Сохранить в любую удобную папку.
2) После завершения загрузки распаковать так же в любое удобное место.
3) Открыть 1С-Предприятие 8.3, создать базу, добавить конфигурацию из указанного места, указать на место распаковки архива.

## Для работы сервиса необходимо сторонне програмное обеспечение
4) Скачать Apache ver 2.2.* в виде msi installer. Необходимо это для того, что бы платформа 1с видела серверы как объекты приложений(об этом подробнее в пояснительной записке)
5) Установить, проверить доступность порта 8080 для Apache (иногда может заниматься скайпом или различными сетевыми приложениями). В случае занятости - при установке указать апачу другой порт, либо же переставить занимающее приложение на другой.
6) Запустить Apache Monitor, в нём должен быть запущен уже один сервер. Проверить статус, должен быть Started.
7) В браузере перейти на localhost, увидеть что всё работает (именно так там будет и написано).
8) Скачать SoapUI - программу для тестирования веб-сервисов. Официальный сайт в гугле по 1 ссылке, soap-ui.org.
9) Установить её, запустить, создать пустой проект, свернуть.
10) В системе 1С запустить конфигуратор, и опубликовать конфигурацию на веб-сервер. (Администрация, публикация на веб-сервер).
11) Перейти по адрессу http://localhost/server/ws/getDocumentByName?wsdl, и увидеть большой список xml записей. Почему именно этот адресс - подробнее в конфигурации.
12) В SoapUI нажать правой кнопкой мыши по проекту, выбрать пункт Add WSDL, и в появившейся форме так же добавить эту ссылку. Нажать ок, в дереве проектов появится имя сервиса.
13) Развернуть имя сервиса, внутри будет Request 1 - открыть двойным щелчком ЛКМ, появится шаблон запроса к сервису, вместо знака ? поставить 1 или 2, справа появится результат запроса.

Более подробно каждый шаг и всё связанное с ним будет описано в общей пояснительной записке, со скриншотами.
